# get-ocp-version

Tekton task to collect OCP version tag from FBC fragment using `skopeo inspect`.

## Parameters

| Name        | Description           | Optional | Default value |
|-------------|-----------------------|----------|---------------|
| fbcFragment | A FBC container Image | No       | -             |
